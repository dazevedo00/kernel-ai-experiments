Processamento Automático de Recursos Humanos
URL: http://127.0.0.1:2018/WebApi/RecursosHumanos/Processamento/EfectuaProcessamentoAutomatico/{Ano}/{nPeriodo}/{MesAProcessar}/{TipoPeriodo}/{DiasProc}/{DataFimPeriodo}/{DataProc}/{strRestricoesFuncs}/{mensagem}
Description: Esta API realiza um processamento automático de recursos humanos.
Operation: POST
Parameters:
- Ano: O ano para o qual o processamento deve ser realizado. (integer, Required)
- nPeriodo: O número do período para o qual o processamento deve ser realizado. (integer, Required)
- MesAProcessar: O mês que será processado. (integer, Required)
- TipoPeriodo: O tipo de período a ser processado. (string, Required)
- DiasProc: O número de dias a serem processados. (double, Required)
- DataFimPeriodo: A data de término do período. (date-time, Required)
- DataProc: A data de processamento. (date-time, Required)
- strRestricoesFuncs: Restrições de funcionários. (string, Required)
- mensagem: Mensagem relacionada ao processamento. (string, Required)
